## 项目简介
PathForge 是面向 MoonBit 生态的轻量级图搜索与网格寻路工具库，可用于游戏 AI、地图导航、路线规划、算法教学和 WebAssembly 可视化。项目提供 `Graph`、`Grid`、`Point`、`PathReport` 等基础模型，支持 BFS、双向 BFS、Dijkstra、A*、地形权重、可达性检查和可达节点统计。

## 核心功能范围
提供邻接表图模型、二维网格地图、障碍物设置、地形移动代价、路径搜索算法、统一结果返回结构、MoonBit 单元测试和 README 使用示例。

## 原创或参考说明
本项目为原创项目，不是对其他语言开源库的移植。项目仅参考 BFS、Dijkstra、A*、双向 BFS 等经典公开算法思想，代码实现、API 设计和测试用例均围绕 MoonBit 重新设计。项目拟采用 MIT License 开源。
